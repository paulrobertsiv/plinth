Python egg:
Download the egg file:
         http://antlr.org/download/Python/
         http://antlr.org/download/Python//antlr_python_runtime-3.1.2-py2.5.egg
Install using easy_install-2.5

Antlr 3.1.2:
Download Antlr 3.1.2 and extract the file:
         http://www.antlr.org/download.html   
         http://www.antlr.org/download/antlr-3.1.2.tar.gz
Move the directory to /usr/lib/
Add all the files in antlr*/lib/ to your CLASSPATH (see pg. 10 of book)

You're all set!
